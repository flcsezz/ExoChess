# Proposed New Features

## 1. Opening Trainer (Spaced Repetition)
Leverage the existing `assets/chess_openings.db` to create an active recall trainer where users can play against the engine specifically within a chosen opening line. This would help users learn opening theory through practice rather than just memorization.

## 2. PGN/FEN Share Extension
Implement a native iOS/Android share extension. This allows users to "share" PGN or FEN strings from external sources (browsers, other chess apps, or emails) directly into the application's analysis board, making it easier to analyze games on the go.

## 3. Pass-and-Play with Integrated Clock
Combine the existing `ClockTool` and OTB (Over-the-Board) mode into a single unified "Pass-and-Play" experience. This would turn the app into a complete solution for playing over-the-board games with friends, providing a professional chess clock interface.

## 4. Post-Puzzle Engine Analysis
In the `PuzzleScreen`, allow users to toggle the Stockfish engine on or off after they've completed or failed a puzzle. This gives users immediate feedback and the ability to explore why a specific move was wrong or what the best line was without needing to copy the FEN to a separate analysis screen.
